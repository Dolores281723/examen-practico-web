Mis Servicios

Como desarrolladora web en formación, ofrezco una serie de servicios para ayudarte a tener presencia en internet. Pongo todo mi esfuerzo y dedicación en cada proyecto para que el resultado sea de la mejor calidad posible.

¿Qué puedo ofrecerte?

  • Creación de sitios web estáticos

    ¿Necesitas una página de presentación, un portfolio o una web sencilla para tu negocio? Puedo crearla desde cero utilizando HTML y CSS, asegurando un diseño limpio y una estructura clara.

  • Mantenimiento web

    Si ya tienes una web pero necesitas actualizar su contenido, corregir pequeños errores o hacer cambios puntuales, puedo encargarme de ello para que tu sitio esté siempre al día.

  • Diseño responsive

    Hoy en día es fundamental que una web se vea bien en cualquier dispositivo. Me aseguro de que todos mis diseños sean adaptables a móviles, tablets y ordenadores de escritorio.

  • Optimización básica

    Para que tu web cargue más rápido y sea más amigable para los buscadores, aplico técnicas de optimización de imágenes y código.
